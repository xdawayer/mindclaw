# Translate

## Steps
1. Detect the source language
2. Translate to the target language specified by the user
3. Preserve original formatting (markdown, code blocks, etc.)
4. If no target language is specified, default to English

## Notes
- For long documents, translate section by section
- Maintain technical terms in their original form when appropriate
